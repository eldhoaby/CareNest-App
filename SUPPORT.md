# 💬 Support

We are dedicated to providing support for the CareNest platform. If you encounter any issues or need assistance, please follow the guidelines below.

## How to Get Help

### 1. Check the Documentation
Before reaching out, please explore our comprehensive [documentation](./docs/README.md). Many common issues regarding setup, deployment, and configuration are answered there.

### 2. Search Existing Issues
Check the [GitHub Issues](https://github.com/eldhoaby/CareNest-App/issues) to see if someone has already reported your problem. If you find an existing issue, you can subscribe to it or add your findings.

### 3. Open a New Issue
If you can't find an answer, please open an issue:
- **For Bugs:** Use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) template and provide as much detail as possible, including logs and reproduction steps.
- **For Features:** Use the [Feature Request](.github/ISSUE_TEMPLATE/feature_request.md) template.

### 4. Security Vulnerabilities
> [!WARNING]  
> Please **do not** report security vulnerabilities via public GitHub issues. 

If you discover a security vulnerability, please refer to our [SECURITY.md](SECURITY.md) for instructions on how to responsibly disclose it.

## Community & Discussions
For general questions, architecture discussions, or sharing your deployment setup, feel free to use [GitHub Discussions](https://github.com/eldhoaby/CareNest-App/discussions) (if enabled) or contact the maintainers directly.
