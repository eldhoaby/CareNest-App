# Screenshots

This directory contains screenshots and visual assets for the CareNest project repository.

## Adding New Screenshots
When adding new UI screenshots, please ensure they are:
1. High resolution (preferably 1080x1920 for mobile).
2. Saved in `.png` format.
3. Named descriptively (e.g., `caregiver_dashboard.png`, `elderly_alerts.png`).
4. Compressed to minimize repository bloat.
