### Added
- Core mobile application built with Flutter.
- Real-time Firestore synchronization for vital signs and environmental sensors.
- Role-based interfaces for Elderly, Caregivers, and Admins.
- Push notification support for critical alerts.
- Backend infrastructure integrated with React Admin Portal.
